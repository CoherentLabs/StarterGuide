When you need to render a rotating 3D character portrait, a 3D minimap, or a live item preview, embedding heavy video files isn't the answer. Live Views bridge the gap.

## Topics

* **What are Live Views?**
    * **Context:** Provide a high-level explanation of how the C++ engine can render a 3D scene (like a character model) directly into a 2D UI texture in real-time.
* **The Frontend Integration:**
    * **Context:** Explain how the frontend developer hooks into this feature. Usually, this involves using specific `src` attributes or CSS `background-image` paths provided by the C++ team to display the live render target directly in an `<img>` or `<div>`.
