Take your UI to the next level by leveraging Gameface's advanced rendering capabilities for complex visual effects.

## Topics

* **Backdrop Filters:**
    * **Context:** Explain how to use the CSS `backdrop-filter` property to create frosted glass effects or blurs over the 3D game world directly behind the UI panel. Note the performance cost of blurs.
* **Surface Partitioning:**
    * **Context:** Briefly explain what surface partitioning is (grouping distinct UI elements to optimize how the engine updates and draws them) and how frontend HTML/CSS structure influences this C++ grouping for better performance.
